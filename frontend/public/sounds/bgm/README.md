# Background Music (BGM)

This directory contains background music files for Heart Whisper Town.

## Expected Files

- `peaceful-town.mp3` - Main background music for the game

## Adding Background Music

1. Place your MP3 files in this directory
2. The files will be automatically loaded by the sound manager
3. If files are missing, the system will gracefully fall back to silent mode
